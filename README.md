# Math Madness

## Overview
This document describes the requirements for the software implementation of the classic Memory game, with a math twist. Following this software will allow 2 players to play the game of memory together.

## The Game
The players take turns choosing two “cards”, one from each section. When a card from the second section presents the answer to the example on the first card, the match is taken and the player earns a point. Otherwise, when it isn’t a match, the cards get returned to their original positions.
